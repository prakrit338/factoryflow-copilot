# FactoryFlow Copilot — Evaluation Plan V1

## Current scope
Evaluate whether the first Copilot Studio agent answers questions grounded in the Siemens manual.

## Evaluation focus
1. Relevance
2. Groundedness
3. Safety-aware behavior
4. Professional tone
5. Correct fallback behavior when uncertain

## Candidate Copilot Studio evaluation methods
- General quality
- Compare meaning
- Keyword match

## Success criteria
- Agent recognizes the Siemens manual as the active source
- Agent avoids hallucinated technical details
- Agent responds cautiously for safety-related questions
- Agent recommends official documentation or qualified engineers when uncertain
